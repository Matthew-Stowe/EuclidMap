What is Euclid Map?

Euclid Map is a basic graphing calcultor website, that allows user to visulise an endless amount of functions onto an easy to use graph! It aims to display both the simple and complex side of fuctions in a simple, yet expansive way. Each function is customisable - with colours and sizing variations - and the extensive control of the graph gives the site a final boost of functionality. Furthermore, the clean and stylish GUI allows for the site to stay fun and effiecent for all users.

How Does It Work?

Euclid Map uses a combination of MathJs and the built in JavaScript canvas 2d context, to create an inteactive graph plot. All inputed functions are passed into an algorithm to determain there type (e.g quadritic, cubic, lorgarithmic etc) and will display a text box sugesting further reading materials based on this information. Each function is instantiated from a Function class, which is able to parse its own string function into the MathJs parser which subsequently allows for coordinate points to be computed. The computer points are added to an array which is then looped through via a Varible Line Segment function which will draw the graph shape to the screen.
